# Tier B avatar 素材 (Sprint 6-3)

在本目录放三段视频, 文件名固定:

| 文件 | 内容 | 建议时长 |
|---|---|---|
| `idle.mp4` | 聆听: 眨眼、轻微点头、身体微动 | 6-10s 可循环 |
| `talking.mp4` | 说话: 口型开合、自然手势 | 6-10s 可循环 |
| `thinking.mp4` | 思考: 低头看资料 / 点头沉吟 | 4-8s 可循环 |

## 生成规范

- **同一人物、同一机位、同一光线**——三段切换时不能"换人"。
  用 HeyGen / 可灵 / 即梦等生成时固定 seed / 用同一张参考人像。
- 正面面对镜头, 半身近景, 背景简洁 (办公室虚化最稳)。
- **首尾帧尽量一致**(循环点不跳变): 生成后可用 ffmpeg 裁到动作闭环处。
- 格式: mp4 (H.264 + 无音轨), 720p (1280x720), ≤3MB/段为宜:
  ```bash
  ffmpeg -i in.mp4 -an -vf scale=1280:720 -c:v libx264 -crf 26 -movflags +faststart idle.mp4
  ```
- 合规: 人物必须是 AI 生成或已获肖像授权; UI 已有「AI 虚拟面试官」标识,
  素材本身不需要再打水印。

## 行为

- 素材缺失 / 加载失败: 前端自动退回占位面板 (人形图标), 面试不受影响。
- Tier A (LiveTalking + MuseTalk 真口型) 接入后本目录素材退役,
  但 idle/talking/thinking 三态语义保留。
